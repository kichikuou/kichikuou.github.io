鬼畜王 on Chrome

鬼畜王ランス以外のゲームをインストールする

鬼畜王 on Chrome は xsystem35 を利用しているため、xsystem35 で動くゲームなら動かせる可能性があります。

上記のリストにあるゲームでも、DOS/Windows3.1 用と Windows95 用がある場合は、Windows95 用（＝ System3.5 版）しかインストールできません。例えばアリスソフトアーカイブズで配布されている「マルチプラットフォーム版」は System3 版なのでインストールできません。

同時に複数のゲームをインストールすることはできません。別のゲームをインストールするには先にアンインストールしてください。（セーブデータのバックアップをお忘れなく）

ディスクイメージからインストールする

CloneCD 形式 (.img + .cue) または Alcohol 120% 形式 (.mdf + .mds) のイメージがあれば、通常のインストーラにドラッグ＆ドロップでインストールできます。

ファイルからインストールする

ゲームの CD をお持ちの場合は、CD 内のファイルを使ってインストールできます。

ただし CD音源の BGM はブラウザからは読み出す方法がないので、Windows Media Player などを使ってオーディオトラックをファイルとして取り込んでいただく必要があります。ファイル形式は wav, mp3, ogg に対応しています。このインストーラでは、ファイル名の拡張子直前の部分をトラック番号として認識します。つまり、02.wav, 03.wav, 04.wav, ... や、 Track2.mp3, Track3.mp3, Track4.mp3, ... のようなファイル名で取り込んでください。

以下のファイルをすべてこのページにドラッグ＆ドロップして、ページ一番下の「インストール」ボタンを押してください。

  • ゲーム CD の GAMEDATA フォルダ内の拡張子 .ALD のファイル
  • CD のオーディオトラックを取り込んだ音楽ファイル

ゲームごとのインストール方法

Rance－光を求めて－ (アリスの館456収録版)

配布サイトから「移植456」をダウンロード・展開して、Rance1 フォルダ内の .ALD ファイルをすべてこのページにドラッグ＆ドロップします。

配布サイトから「移植456」をダウンロード・展開して、Rance2 フォルダ内の .ALD ファイルをすべてこのページにドラッグ＆ドロップします。

配布サイトから「移植456」をダウンロード・展開して、「闘神都市」フォルダ内の .ALD ファイルをすべてこのページにドラッグ＆ドロップします。
