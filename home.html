詳しい説明

アリスソフトのゲームエンジン System3.5 の有志によるフリー実装である xsystem35 を Portable Native Client に移植したものです。Windows, Mac, Linux, Chrome OS 上の Chrome で動きます。 Android と iOS 版 Chrome では動作しません。

素晴らしいゲームがせっかくフリーで提供されているのに、遊ぶためのハードルが高い（CD-R に書き込むか仮想 CD ソフトを入れるか、Windows 以外ではもっと大変…）という状況を改善できれば、という試みです。

このサイト自体は実行環境のみ提供するもので、遊ぶためにはゲームの CD イメージを別途ダウンロードする必要があります。（インストール ページに配布サイトへのリンクがあります）

ソースコードは以下の場所にあります。

  • https://github.com/kichikuou/xsystem35-nacl - xsystem35 の PNaCl 版
  • https://github.com/kichikuou/kichikuou.github.io - このサイト
