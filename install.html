インストール手順

(注意) 18歳未満の人は利用できません。

ゲームを遊ぶには、CDイメージをダウンロードして（Chrome上に）インストールする必要があります。

  1. 以下のいずれかのサイトで、鬼畜王ランスの CDイメージをダウンロードします。
    • アリスソフト アーカイブズ
    • 鬼畜王ランス ミラー配布所
    • アリスソフト『配布フリー宣言』対象ソフト配布
  2. ダウンロードしたファイルは RAR 形式の圧縮ファイルなので、展開して CD-ROM イメージを取り出します。 Windows なら LhaPlus、 Mac なら The Unarchiver で展開できます。 ChromeOS は標準機能で RAR を展開できます。
  3. 展開したフォルダにある KICHIKUOU.img と KICHIKUOU.cue （または DISC.img と DISC.cue） を、このウィンドウにドラッグ＆ドロップするか、下の「ファイルを選択」ボタンを押して選択します。
  4. ファイル保存の許可を求めるポップアップが表示されたら、「許可」をクリックします。（CD-ROM イメージから取り出したファイルを保存するために必要です。）約 650MB の空きディスク容量が必要です。
  5. インストールが始まります。インストール完了後は CD イメージを削除しても構いません。
